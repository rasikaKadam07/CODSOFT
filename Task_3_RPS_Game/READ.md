## Rock Paper Scissors Game

This is my first project during the "CodSoft Internship" in python programming. 
It's a fully-featured Rock Paper Scissors game with an AI opponent.

## Features
- 3 difficulty levels (Easy, Medium, Hard)
- Best of 3, 5, or 10 match modes
- Score tracking
- Game statistics dashboard
- Beautiful colored output

## Requirements
- Python 3.8 or higher
- Rich library

## How to Run
1. **Install the required library:**
   ```bash
   pip install rich

## run the game
    python rps_game.py

## How to play?
- Choose play match
- Select Game Mode(best_of_3/best_of_5/best_of_10)
- Select Difficulty(easy/medium/hard)
- Choose rock, paper, or scissors
- Beat the AI to win points
- First to win set number of rounds wins match

## Example GamePlay
    Round 1
    Score: You 0 - 0 AI

    Your choice (rock, paper, scissors): rock
    AI chose: scissors
    ✓ You win this round!
